## Usages
1. Add it in your root build.gradle at the end of repositories:
```Groovy
allprojects {
    repositories {
        ...
        maven { url 'https://jitpack.io' }
    }
}
```
2. Add the dependency
``` Groovy
dependencies {
    implementation 'com.github.Lulixue:FlexibleContainerView:1.0'
}
```
